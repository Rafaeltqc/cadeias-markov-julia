# Cadeias de Markov em Julia

Projeto acadêmico desenvolvido para resolver problemas envolvendo cadeias de Markov utilizando a linguagem Julia.

## Objetivo

Aplicar conceitos de probabilidade para modelar sistemas que mudam de estado com determinadas probabilidades de transição.

## Conceitos utilizados

- Cadeias de Markov
- Matrizes de transição
- Probabilidade
- Processos estocásticos

## Linguagem utilizada

- Julia
